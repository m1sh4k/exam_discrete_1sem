# Логика предикатов. Понятие предиката. Кванторные операции. Общность высказываний. Формализация общих и частных высказываний.
## Определения


**Предикат** — функция, принимающая значения истинности.

**Область определения** предиката называется **миром речи.**

$P(x)$
- $P$ ­— предикат _(функция)_
- $x$ — переменная
- $P(x)$ — высказывание

Если все переменные пробегают **один и тот же** набор объектов, то говорят, что мир речи **односортный**.

Если различные переменные пробегают **различные** наборы объектов, то говорят, что мир речи **многосортный**.

### Отличие от исчисления высказываний

| Аспект                                   | Предикаты       | Высказывательные функции |
| ---------------------------------------- | --------------- | ------------------------ |
| Область определения                      | Любое множество | Значение истинности      |
| Внутреннее строение простых высказываний | Рассматривается | **НЕ** рассматривается   |

Логическое высказывание из предиката можно получить только при подстановке вместо его переменных  конкретных значений из его области определения. 

## Область истинности

Область истинности предиката — совокупность всех объектов мира речи, для которых предикат принимает значение «истина».
Обозначается как $I_{p}$ .

Для объектов мира речи, **НЕ** входящих в область истинности, предикат принимает значение «ложь».

Атомарные предикаты могут быть комбинироваться в более сложные.
_Т. к. предикаты возвращают значения истинности, с использованием операций исчисления высказываний_

## Кванторы и формализация

### Существование

$\exists$ — 'exists'
Пусть предикат $P(x)$ определен на множестве $I$
Тогда $\exists P(x)$ истинно, когда существует элемент $a\in I$ для которого $P(a)$ — истинна

#### Пример формализации

_"Некоторые S есть P"_

1. Вводим предикаты
   $S(x)$ — "$x$ есть $S$"
   $P(x)$ — "$x$ есть $P$"
2. Ставим квантор, предикаты, с операцией " $\land$ "
   $\exists x[S(x)\cdot P(x)]$
   
### Всеобщность

$\forall$ — "all"

Пусть предикат $P(x)$ определен на множестве $I$
Тогда $\exists P(x)$ истинно, когда для всех элементов $a\in I$ , $P(a)$ — истинна

#### Пример формализации

_"Все S есть P"_

1. Вводим предикаты
   $S(x)$ — "$x$ есть $S$"
   $P(x)$ — "$x$ есть $P$"
2. Ставим квантор, предикаты с операцией " $\to$ "
   $\forall x[S(x)\to P(x)]$
   
## Связь частных и общих высказываний

$\exists[P(x)]=\neg \forall[\neg P(x)]$
$\forall[P(x)]=\neg \exists[\neg P(x)]$
## Деление высказываний по степени общности 

1. Единичные _(о конкретных примерах)_
2. Множественные
   - Общие _(о всех предметах некоторого класса)_
   - Частные _(о некоторых предметах класса)_
