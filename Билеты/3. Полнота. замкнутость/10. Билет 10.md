# Класс монотонных функций и его замкнутость. Примеры проверки функций на монотонность.

## Определение монотонных функций (класс $M$)

$\forall i(a_{i}\preceq b_{i})\implies f(a_{1},\dots,a_{n})\preceq f(b_{1},\dots,b_{n})$ — значит $f$ монотонна

### Доказательство монотонности

Пусть
$\forall i(a_{i}\preceq b_{i})\implies f(a_{1},\dots,a_{n})\preceq f(b_{1},\dots,b_{n})$ — $f$ монотонна

#### Отождествление 
Заменив $k$-й элемент на $j$-й, на $k$-м элементе получим пару $a_{j},\ b_{j}$, по свойству функции $a_{j}\preceq b_{j}$, монотонность сохраняется.

#### Подстановка
Если $j$-й элемент заменить на монотонную функцию $g$, то $\forall i(a_{i}\preceq b_{i})\implies g(a)\preceq g(b)\implies$ $j$-й элемент в первом наборе $\preceq$ во втором, монотонность сохраняется


## Пример

### Принадлежат
$a\land b$ 

| $a$ | $b$ | $a\land b$ | Выполняется  |
| --- | --- | ---------- | ------------ |
| 0   | 0   | 0          | $\checkmark$ |
| 0   | 1   | 0          | $\checkmark$ |
| 1   | 0   | 0          |              |
| 1   | 1   | 1          | $\checkmark$ |

### Не принадлежат
$a\oplus b$


| $a$ | $b$ | $a\oplus b$ | Выполняется  |
| --- | --- | ----------- | ------------ |
| 0   | 0   | 0           | $\checkmark$ |
| 0   | 1   | 1           | $\times$     |
| 1   | 0   | 1           |              |
| 1   | 1   | 0           |              |
