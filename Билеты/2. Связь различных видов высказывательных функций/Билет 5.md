## Определения 

- **Ранг** элементарной конъюнкции/дизъюнкции — количество первичных термов, из которых она состоит.
## Дизъюнктивная нормальная форма

**ДНФ** — представление высказывательной функции в виде формулы, являющейся дизъюнкцией элементарных конъюнкций, которая:
- Не содержит одинаковых элементарных конъюнкций
- Не содержит элементарные конъюнкции, отличающиеся только порядком следования первичных термов

**Слагаемые ДНФ** — элементарные конъюнкции, из которых состоит ДНФ.
**Длина ДНФ** — число слагаемых ДНФ.
**Сложность ДНФ** — сумма рангов слагаемых ДНФ.
### Виды ДНФ

1. **Совершенная** — все элементарные конъюнкции — конституенты единицы.
2. **Минимальная** — содержит наименьшее число вхождений первичных термов среди всех ДНФ _(наименьшая сложность)_
3. **Кратчайшая** — включает в себя наименьшее число элементарных конъюнкций среди равносильных ДНФ _(наименьшее число слагаемых)_
4. **Сокращенная** _(СокрДНФ)_ — состоит из простых импликант функции
5. **Тупиковая** — сокращенная ДНФ, в которой отбрасывание любой из простых импликант приводит к неравносильной ДНФ

## Связь с таблицей истинности

Пусть есть функция $f(x_{1},\dots,x_{n})$, принимающая $n$ аргументов

### Получение СДНФ из ТИ

Смотрим строки, где значение функции равно 1, строим конституент единицы:

$\bigwedge_{i=1}^{n}\pm x_{i}$
- $x_{n}$ если в данной интерпретации $x_{n}=1$
- $\overline{x_{n}}$ если в данной интерпретации $x_{n}= 0$

Пусть у нас $m$ конституентов k

тогда $f(x_{1},\dots ,x_{n})=k_{1}\lor k_{2}\lor\dots \lor k_{m}$

У противоречия нет СДНФ, т.к. нет значений 0 и не из чего строить.
### Получение СКНФ из ТИ


Смотрим строки, где значение функции равно 0, строим конституент нуля:

$\bigvee_{i=1}^{n}\pm x_{i}$
- $x_{n}$ если в данной интерпретации $x_{n}=0$
- $\overline{x_{n}}$ если в данной интерпретации $x_{n}= 1$

Пусть у нас $m$ конституентов k

тогда $f(x_{1},\dots ,x_{n})=k_{1}\land k_{2}\land\dots \land k_{m}$

У тавтологии нет СКНФ, т.к. нет значений 1 и не из чего строить.
