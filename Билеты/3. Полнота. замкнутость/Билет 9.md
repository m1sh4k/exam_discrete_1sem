# Классы функций, сохраняющих константы и их замкнутость. Примеры проверки функций на сохранение констант

## Сохраняющие 0 $(T_{0})$
$f(0,\dots,0)=0$

### Доказательство монотонности

При подстановке все равно останется 0, при замене 0 не поменяется. При всех аргументах 0 функция в любом случае будет давать 0.

### Примеры
#### Принадлежат
$0\land0=0$
$0+0=0$
$0\oplus0=0$

#### Не принадлежат
$0\to0=1$
$\neg0=1$
## Сохраняющие 1 $(T_{1})$
$f(1,\dots,1)=1$

### Доказательство монотонности

При подстановке все равно останется 1, при замене 1 не поменяется. При всех аргументах 1 функция в любом случае будет давать 1.

### Примеры

#### Принадлежат
$1\land1=1$
$1+1=1$
$1\to1=1$

#### Не принадлежат
$1\oplus1=0$
$\neg1=0$
