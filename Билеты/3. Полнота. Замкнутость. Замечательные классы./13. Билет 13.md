#  Критерий функциональной полноты Поста-Яблонского. Примеры применения критерия к проверке систем функций на полноту.

## Критерий функциональной полноты Поста-Яблонского


Система высказывательных функций функционально **полна**, если она полностью не лежит ни в одном из предполных классов.

Класс функционально полон, если:
- содержит хотя бы одну функцию $f_{1}$, не принадлежащую классу [ T₀](%D0%91%D0%B8%D0%BB%D0%B5%D1%82%D1%8B/3.%20%D0%9F%D0%BE%D0%BB%D0%BD%D0%BE%D1%82%D0%B0.%20%D0%97%D0%B0%D0%BC%D0%BA%D0%BD%D1%83%D1%82%D0%BE%D1%81%D1%82%D1%8C.%20%D0%97%D0%B0%D0%BC%D0%B5%D1%87%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B5%20%D0%BA%D0%BB%D0%B0%D1%81%D1%81%D1%8B./9.%20%D0%91%D0%B8%D0%BB%D0%B5%D1%82%209.md)
- содержит хотя бы одну функцию $f_{2}$, не принадлежащую классу [T₁](%D0%91%D0%B8%D0%BB%D0%B5%D1%82%D1%8B/3.%20%D0%9F%D0%BE%D0%BB%D0%BD%D0%BE%D1%82%D0%B0.%20%D0%97%D0%B0%D0%BC%D0%BA%D0%BD%D1%83%D1%82%D0%BE%D1%81%D1%82%D1%8C.%20%D0%97%D0%B0%D0%BC%D0%B5%D1%87%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B5%20%D0%BA%D0%BB%D0%B0%D1%81%D1%81%D1%8B./9.%20%D0%91%D0%B8%D0%BB%D0%B5%D1%82%209.md)
- содержит хотя бы одну функцию $f_{3}$, не принадлежащую классу [L](%D0%91%D0%B8%D0%BB%D0%B5%D1%82%D1%8B/3.%20%D0%9F%D0%BE%D0%BB%D0%BD%D0%BE%D1%82%D0%B0.%20%D0%97%D0%B0%D0%BC%D0%BA%D0%BD%D1%83%D1%82%D0%BE%D1%81%D1%82%D1%8C.%20%D0%97%D0%B0%D0%BC%D0%B5%D1%87%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B5%20%D0%BA%D0%BB%D0%B0%D1%81%D1%81%D1%8B./12.%20%D0%91%D0%B8%D0%BB%D0%B5%D1%82%2012.md)
- содержит хотя бы одну функцию $f_{4}$, не принадлежащую классу [M](%D0%91%D0%B8%D0%BB%D0%B5%D1%82%D1%8B/3.%20%D0%9F%D0%BE%D0%BB%D0%BD%D0%BE%D1%82%D0%B0.%20%D0%97%D0%B0%D0%BC%D0%BA%D0%BD%D1%83%D1%82%D0%BE%D1%81%D1%82%D1%8C.%20%D0%97%D0%B0%D0%BC%D0%B5%D1%87%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B5%20%D0%BA%D0%BB%D0%B0%D1%81%D1%81%D1%8B./10.%20%D0%91%D0%B8%D0%BB%D0%B5%D1%82%2010.md)
- содержит хотя бы одну функцию $f_{5}$, не принадлежащую классу [S](%D0%91%D0%B8%D0%BB%D0%B5%D1%82%D1%8B/3.%20%D0%9F%D0%BE%D0%BB%D0%BD%D0%BE%D1%82%D0%B0.%20%D0%97%D0%B0%D0%BC%D0%BA%D0%BD%D1%83%D1%82%D0%BE%D1%81%D1%82%D1%8C.%20%D0%97%D0%B0%D0%BC%D0%B5%D1%87%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B5%20%D0%BA%D0%BB%D0%B0%D1%81%D1%81%D1%8B./11.%20%D0%91%D0%B8%D0%BB%D0%B5%D1%82%2011.md)
