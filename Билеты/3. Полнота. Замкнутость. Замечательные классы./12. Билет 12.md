# Класс линейниых по Жегалкину функций и его замкнутость. Примеры проверки функций на линейность.

## Линейные функции (класс $L$)

### Определение

_Представимы в виде линейного полинома Жегалкина первой степени, т.е._ $c_{0}\oplus a_{1}c_{1}\oplus a_{2}c_{2}\oplus\dots \oplus a_{n}c_{n}$  _(коэффициент перед произведением с более чем одной переменной равен нулю, каждая переменная встречается не более одного раза)_

**Полином Жегалкина** — Жегалкинская сумма попарного произведения переменных и константы определенной для каждого произведения, где ни одна переменная не встречается в одном произведении более одного раза.)
### В общем виде выглядит

$f(x_{1},\ldots ,x_{n})=x_{1}\oplus x_{2}\oplus\dots \oplus x_{n}$ причем не обязательно все $x_{i}:i\in[1;n]$ есть в выражении.

## Доказательство замкнутости

### Отождествление

$f(x_{1},\dots x_{j},\dots,x_{k},\dots ,x_{n}):=c_{0}\oplus c_{1}x_{1}\oplus\dots \oplus c_{j}x_{j}\oplus\dots \oplus c_{k}x_{k}\oplus\dots \oplus c_{n}x_{n}$
$h(x):=f(x_{1},\dots x_{j},\dots,x_{j},\dots ,x_{n})=c_{0}\oplus c_{1}x_{1}\oplus\dots \oplus c_{j}x_{j}\oplus\dots \oplus c_{k}x_{j}\oplus\dots \oplus c_{n}x_{n}$
Линейна по Жегалкину.

### Подстановка

$a\oplus a=0$

$f(x_{1},\dots,x_{k},\dots ,x_{n}):=c_{0}\oplus c_{1}x_{1}\oplus\dots \oplus c_{j}x_{j}\oplus\dots \oplus c_{k}x_{k}\oplus\dots \oplus c_{n}x_{n}$
$g(y_{1},\dots y_{n}):=c'_{0}\oplus c'_{1}y_{1}\oplus\dots \oplus c'_{j}y_{j}\oplus\dots \oplus c'_{k}y_{k}\oplus\dots \oplus c'_{n}y_{n}$

$h(x)=f(x_{1},\dots x_{j},\dots,g(y_{1},\dots,y_{n}),\dots ,x_{n})=$
$(c_{0}\oplus c'_{0})\oplus c_{1}x_{1}\oplus\dots \oplus c_{j}x_{j}\oplus\dots \oplus c_{k}x_{k}\oplus\dots \oplus c_{n}x_{n}\oplus \oplus c'_{1}y_{1}\oplus\dots \oplus c'_{j}y_{j}\oplus\dots \oplus c'_{k}y_{k}\oplus\dots \oplus c'_{n}y_{n}$
Линейна по Жегалкину.
